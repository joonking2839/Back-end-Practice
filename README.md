# 축구 구단주 시뮬레이션

##  프로젝트 개요

이 프로젝트는 **구단주가 되어 팀을 운영**하고,  
선수를 영입,교체,방출하며 전술을 설정해 경기를 치르는 축구 시뮬레이션 게임입니다.
게임을 실행하면
선수들의 **기여도(Contribution)**를 기반으로 승부를 예측합니다.

처음 프로그램 실행시 구단주를 생성하고 이적시장에서 11명의 선수를 구입하면서 자신만의 포메이션을 구성 후에 
게임을 실행합니다.
